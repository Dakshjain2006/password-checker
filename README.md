# 🔐 Password Strength Checker

Client-side-only password security analyzer — no data is ever transmitted or stored.

## 🚀 Live Demo
[Dakshjain2006.github.io/password-checker](https://Dakshjain2006.github.io/password-checker)

## ✨ Features
- Real-time 4-tier strength: Weak / Fair / Good / Strong
- 6-criteria checklist (length, uppercase, lowercase, number, symbol, 12+ chars)
- Animated colour-coded progress bars
- Random secure password generator (16 chars, all character classes)
- Show/hide password toggle
- 100% client-side — zero network requests

## 🛠 Tech Stack
HTML5 · CSS3 · Vanilla JavaScript · Regex
